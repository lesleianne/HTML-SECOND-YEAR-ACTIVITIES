📋 Algorithm Overview

Two Recursive Functions:
• findFrequencies() - Main function that processes each unique element
• countFrequency() - Helper function that counts occurrences of a specific element

🎯 Step 1: Processing Element 1

1findFrequencies(arr, 9, 0) - Start at index 0
Current element: arr[0] = 1 (not a duplicate, so process it)
countFrequency(arr, 9, 1, 0) - Count occurrences of element 1
Index 0: arr[0] = 1 ✓ (match) → 1 + countFrequency(..., 1)
Index 1: arr[1] = 2 ✗ (no match) → countFrequency(..., 2)
Index 2: arr[2] = 2 ✗ (no match) → countFrequency(..., 3)
... continues until index 9 (base case)
🎯 Result: 1 appears 1 time(s)

🎯 Step 2: Processing Element 2

2findFrequencies(arr, 9, 1) - Move to index 1
Current element: arr[1] = 2 (not a duplicate, so process it)
countFrequency(arr, 9, 2, 0) - Count occurrences of element 2
Index 0: arr[0] = 1 ✗ (no match) → countFrequency(..., 1)
Index 1: arr[1] = 2 ✓ (match) → 1 + countFrequency(..., 2)
Index 2: arr[2] = 2 ✓ (match) → 1 + countFrequency(..., 3)
Index 3: arr[3] = 3 ✗ (no match) → countFrequency(..., 4)
... continues until index 9 (base case)
🎯 Result: 2 appears 2 time(s)

🎯 Step 3: Skipping Duplicate

3findFrequencies(arr, 9, 2) - Move to index 2
Current element: arr[2] = 2
Previous element: arr[1] = 2
SKIP! This is a duplicate (already processed)
⬇️ Skip to next index

🎯 Step 4: Processing Element 3

4findFrequencies(arr, 9, 3) - Move to index 3
Current element: arr[3] = 3 (not a duplicate, so process it)
countFrequency(arr, 9, 3, 0) - Count occurrences of element 3
Only finds 1 occurrence at index 3
🎯 Result: 3 appears 1 time(s)

🎯 Step 5: Processing Element 4

5findFrequencies(arr, 9, 4) - Move to index 4
Current element: arr[4] = 4 (not a duplicate, so process it)
countFrequency(arr, 9, 4, 0) - Count occurrences of element 4
Finds matches at indices 4, 5, and 6
🎯 Result: 4 appears 3 time(s)

🎯 Steps 6-7: Skipping More Duplicates

6findFrequencies(arr, 9, 5) - Skip (duplicate 4)
7findFrequencies(arr, 9, 6) - Skip (duplicate 4)

🎯 Step 8: Processing Element 5

8findFrequencies(arr, 9, 7) - Move to index 7
Current element: arr[7] = 5 (not a duplicate, so process it)
countFrequency(arr, 9, 5, 0) - Count occurrences of element 5
Finds matches at indices 7 and 8
🎯 Result: 5 appears 2 time(s)

🎯 Final Steps

9findFrequencies(arr, 9, 8) - Skip (duplicate 5)
10findFrequencies(arr, 9, 9) - Base case reached (index >= size)

📊 Final Output

1 appears 1 time(s)
2 appears 2 time(s)

4 appears 3 time(s)
5 appears 2 time(s)

🔑 Key Recursive Concepts

Base Cases:
• findFrequencies(): when index >= array size
• countFrequency(): when index >= array size

Recursive Reduction:
• Each call processes one index and moves to the next
• Problem size decreases with each recursive call

Efficiency Optimization:
• Skips duplicate elements since array is sorted
• Only processes each unique element once
